# Results

Store evaluation artifacts here. Keep raw outputs and reports together so later research notes can point to the exact evidence.

Recommended layout:

```text
results/
├── YYYY-MM-DD-agent-name-benchmark/
│   ├── task.json
│   ├── agent_output.json
│   ├── report.json
│   ├── logs/
│   └── notes.md
```

Do not commit private credentials, proprietary simulator files, or large raw traces unless the project policy allows them.
